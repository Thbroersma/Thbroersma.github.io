Werken met CSS

Introductie

Het werken met CSS is erg leuk en ook best makkelijk. Bij HTML werken we met tags <p></p>, met CSS werken we met stijlingtypes.
Daarvoor kies je het element wat je een bepaalde stijl, wilt geven, dan welk styling type en kies je er een. Klinkt weer als een heleboel.
Geen paniek, we gaan er weer stap voor stap doorheen. We gaan de volgende styling types doen:

  • Lettertype - Font-family
  • Breedte/hoogte - Width/height
  • Kleur - Color
  • Uitlijning - Text-align

CSS

Voordat we er mee beginnen, geef ik vast een voorbeeld met een styling en wat meer info over het selecteren van een element.
Bij CSS kies je een element (h1, h2, p, img) waarna je iets met de styling gaat doen. Dat doe je bijvoorbeeld zo:
p {
color:green;
}

En dan wordt de tekst groen. Bij programmeren werk je in het Engels, dus denk hiervast over na voor een aantal kleuren.

Let er op dat je na het genoemde stylingstype eerst : doet en eindigd met een ; . Hiermee sluit je het stylingselement.
Alle stijlingtypes schrijf je met kleine letters, alleen wanneer je een stijlingtype invult kan dat verschillen.
Voor de : altijd kleine letters, na de : alleen beginnende hoofdletter bij een aantal lettertypes.

Lettertype

Na het maken van een tekst is het leuk om er een twist aan te geven. Met 'Font-family' krijg de tekst een andere
manier van schrijven. Er zijn heel veel verschillende types. We zullen met een paar beginnen en zo doorgaan naar wat andere.
Bij het echte programmeren krijg je ook meerdere opties weergegeven, dus dat scheelt. Plus, je kunt altijd wat opzoeken.
Het werken met 'Font-family' gaat het alsvolgt:
p {
font-family: cursive;
}

Breedte/hoogte

Breedte en hoogte is begrijpelijk wat het doet op een pagina. Wanneer gebruik je het alleen? Het aanpassen hiervan is
bijvoorbeeld handig bij afbeeldingen. Zo kun je ermee meerdere afbeeldingen naast elkaar krijgen of naast de tekst krijgen.
Het stellen van een breedte/hoogte doe je met px (pixels), zo is een gemiddelde afbeelding 300px a 400px. Zoals hieronder:
img {
width: 400px
}

De afbeeldingen hebben zo dezelfde breedte en zie je dat er qua breedte nog een afbeelding bij zo kunnen.


Kleur

Het werken met kleur op een website is erg leuk. Je kunt heel gevarieerd hiermee aan de gang gaan. Er zijn alleen een paar
dingen om rekening mee te houden:

  • Dat de tekst in een kleur wel goed leesbaar moet zijn
  • Zo werkt dit bijvoorbeeld niet
  • En dat het past binnen de achtergrond kleur
  • Dat het je de kleur goed spelt in het Engels. Bij twijfel zoek het op (of vraag het)

Dit is een goed passende kleur:

p {
color: orange;
}

Uitlijning

Uitlijnen van de tekst of een onderdeel is handig als je het graag op een bepaalde plek wilt hebben. Alleen wanneer is
dat nou handig? Het kan gebruikt worden wanneer je de tekst in het midden wilt hebben of het ene onderdeel links
en het andere rechts. Dit zijn een aantal voorbeelden van hoe het eruit kan zien:

p {
text-align: right;
}

Ik bleef graag aan de rechterkant

p {
text-align:center
}

Maar kies geen kant en blijf in het midden

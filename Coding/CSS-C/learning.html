Werken met CSS

Introductie

Het werken met CSS is erg leuk en ook best makkelijk. Bij HTML werken we met tags <p></p>, met CSS werken we met styling types.
Daarvoor kies je het element wat je een bepaalde style wilt geven, dan welk styling type en kies je er een. Klinkt weer als een heleboel.
Geen paniek, we gaan er weer stap voor stap doorheen. We gaan de volgende styling types doen:

  • Lettertype - Font-family
  • Breedte/hoogte - Width/height
  • Kleur - Color
  • Uitlijning - Text-align

CSS

Voordat we er mee beginnen, geef ik vast een voorbeeld met een styling en wat meer over selecteren van een element.
Bij CSS kies je een element (h1, h2, p, img) waarna je iets met de styling gaat doen. Dat doe je bijvoorbeeld zo:
p { color:green; }

Lettertype

Titels zijn bekende dingen die je ziet op elke pagina die je bezoekt en zegt vaak iets over wat je kunt vinden op zo'n pagina.
Binnen HTML heb je 6 soorten titels/kopjes die je kunt gebruiken. Ze verschillen in grote en hoe vaak je ze kunt gebruiken op een pagina.

Dieren - <h1></h1>

Huisdieren - <h2></h2>

Honden - <h3></h3>

Golden Retriever - <h4></h4>

Leuke dingen over Golden Retrievers - <h5></h5>
Mijn Golden Retriever Laika - <h6></h6>
Dit zijn de verschillende titels/kopjes die er zijn. Het getal bij elke titel die er staat, is er voor het aantal keer dat die titel
op een pagina mag. Het werken met de titels opvolgende is ook zoals hierboven aangegeven. Bij ieder H2 zal een H3 verder gaan. Verder
in op het betreffende onderwerp.

Breedte/hoogte

Als je tekst onder een titel wilt zetten, kan dat eenvoudig. Dat doe je door de <p></p> tag te gebruiken. Je hebt hier ook een begin <p>
en eind tag </p>, daartussen kun je tekst zetten die zich dan zal weergeven op je website. Als je meerdere regels schrijft, is het handig het alsvolgt
te doen:
<p>
Dit is mijn verhaal over mijn lieve kat Sky.
Sky is een lieve kat die heel erg zacht en aaibaar is.
Helaas is Sky erg op haar eigen moment.
<p>


Kleur

Op een pagina hoort een afbeelding natuurlijk (tenminste op een vrolijke pagina). De afbeelding tag is redelijk eenvoudig, je gebruikt
de img tag, het enige is, het heeft geen eind tag. Als je de img tag gebruikt krijg je het volgende:
<img src="" alt="">
Om een afbeelding te selecteren die je je wilt moet je die noemen in binnen de "" van src.
Als je afbeelding hebt van een kat, dan sla je de afbeelding op als kat.jpg (jpg staat voor het type afbeelding) in je houdige map.
Dan vul je de tag alsvolgt in:
<img src="kat.jpg" alt="">

Uitlijning

Opsommingen zijn handig als je een aantal dingen over een onderwerp wilt opschrijven. Je kunt dit ook makkelijk doen op een website.
Hiervoor heb je alleen 2 soorten tags nodig. De eerste tag is <ul> </ul>, die geeft aan dat je een lijstje met losse punten
wilt gaan maken. Daarbinnen gebruik je <li> </li>, hier binnen schrijf je steeds een punt in de opsomming wilt hebben.
Het zal er zo uit zien:
<ul>
<li>Films kijken</li>
<li>Computerspelletjes spelen</li>
</ul>
Met dit als resultaat:

  • Films kijken
  • Computerspelletjes spelen
